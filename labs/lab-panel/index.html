<lab-panel>

The <lab-panel> webcomponent was created for The Fresno Bee Education Lab landing page. It leverages molecules already installed on the websites and enhances them using the Shadow DOM. The component changes the font face to McClatchy Sans, provides a full-bleed container that can be styled on the page, and an <img> tag included inside the element will both fill the container and be filtered to allow for text to be rendered over it.

Implementation notes for the trailblazers

Custom HTML components require a Javascript class be loaded in order to function. If you want to implement this component in an isolated environment, and ensure that any changes we make going forward do not get incorporated into your story, save a copy of this component and host it wherever you like. We will host the latest version of these files, which may include breaking changes as we iterate. If you would like to automatically get updates, and are comfortable with the risk, you can copy the location below into your embed.
